# Todo Fullstack App

## Live
Frontend: https://todo-fullstack-brown.vercel.app  
Backend: https://todo-backend-xh1o.onrender.com/docs  

## Stack
- FastAPI
- React (Vite)
- PostgreSQL
- Docker
- Render + Vercel

## Features
- rejestracja / login (JWT)
- dodawanie tasków
- oznaczanie jako zrobione
- usuwanie tasków
